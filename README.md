###모스크바 장로교회 성경읽기 기록용 챗봇
---
##개요
---
대청부에서 성경읽기하는거 취합해주는 챗봇임

Cloud.ru에서 무료(는 월 2천원)리소스 대여해서 함

스택은 Sqlite3, flask

언어는 파이썬 사용

개발기간은 주말 포함 1.5일쯤

지피티가 30퍼쯤 써줌 ㅎ

##기능
---
regex 사용해서 단톡방에 올라오는 메시지 파싱함.(유형은 test 파일 참고)

/ 붙이면 명령어인데 뒤에 다른거 못붙임(만들기 구찮음)
- /진행상황 - 요청을 보낸 친구의 몇일차 중 몇일차 완료했는지랑 몇일차 빼먹었는지 보여줌
- /집계 - 디비에 이름 드가있는 사람들 다 보여줌

##문제점
---
1. 유저 닉네임이 바뀌면 안됨. 이전에 했던거 다 날아감(메신저봇 R을 써서 그런듯)
2. /집계가 모든 유저에게 허용되어있음
3. 보안조치 없음
   
문제점 보완할 생각 당장은 없음. 안그래도 바쁨
